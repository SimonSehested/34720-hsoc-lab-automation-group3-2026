**Parameters:**

| Parameter | Default | Description |
|---|---|---|
| `coarse_iterations` | `2` | Number of full coarse sweeps (0→154) per attempt |

| `start_positions` | `None` | `[arm1, arm2, arm3]` start; random if `None` |

**Returns:** `[arm1, arm2, arm3]` — final arm positions (integers, 0–154).

## Algorithm

### Definitions

- **Edge zone:** a position `< 5` or `> 149` (within 5 steps of the hardware limits 0/154).
- **Coarse sweep:** move arm from 0 to 154 while sampling power via a background thread; move arm to the estimated minimum position.
- **Fine refinement:** measure power at `center ± fine_window` (default `fine_window=5`) positions with a brief settle time; expand the window if the minimum is at the boundary; move arm to the true minimum.
- **Attempt:** one full run of all coarse sweeps + all fine refinements.
- **Best result:** across all attempts, the attempt whose final power reading (measured after fine refinement) was lowest.

### Step-by-step

```
attempt = 0
best_result = None   # (positions, power)

loop:
    1. Run coarse sweep for each arm × coarse_iterations
       → record estimated positions

    2. Edge check #1 (after coarse)
       if any arm in edge zone:
           if attempt < max_restarts:
               attempt += 1, randomise start, continue loop
           else:
               goto FALLBACK

    3. Run fine refinement for each arm × fine_iterations
       → measure final power

    4. Measure power at current positions (one stable reading).
       Update best_result if this power < best seen so far.

    5. Edge check #2 (after fine)
       if any arm in edge zone:
           if attempt < max_restarts:
               attempt += 1, randomise start, continue loop
           else:
               goto FALLBACK

    6. No edge issue → return current positions

FALLBACK:
    - Take positions from best_result (if no completed attempt exists,
      use the positions from the current — possibly edge — attempt)
    - Run one final fine refinement (no edge check, no restart)
    - Return resulting positions
```

### Internal helpers (private, not exported)

These are extracted/simplified from `fast_sweep` in `laser_polarization_optimizer.py`:

- `_coarse_sweep(arm_idx, positions)` — sweeps one arm, returns best position estimate
- `_fine_refine(arm_idx, positions, window=5, settle_time=0.1)` — refines one arm, returns best position
- `_get_power()` — reads power meter, returns watts
- `_set_arms(arm1, arm2, arm3)` — sets all three arms
- `_needs_edge_check(positions, margin=5)` — returns True if any arm is in edge zone

All helpers are prefixed with `_` to signal they are internal.

## Behaviour Summary

| Scenario | Behaviour |
|---|---|
| Normal run, no edge positions | Returns after coarse+fine, no restarts |
| Edge after coarse, restarts remaining | Randomises start, full retry |
| Edge after fine, restarts remaining | Randomises start, full retry |
| Cap reached | Takes best positions seen, runs final fine sweep, returns |
| `start_positions=None` | Uses random positions each attempt |
| `start_positions=[a,b,c]` | Uses provided positions on first attempt; random on restarts |
